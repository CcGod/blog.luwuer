---
title: VSCODE常用快捷键
date: 2017-11-14 21:10:26
tags: [VSCODE]
---

```bash
Ctrl + Tab ## 在打开的编辑器（文件）中切换，可以显示打开了哪些文件的列表，然后用鼠标选择

Alt + 上  / 下 ## 上下代码互换位置

Ctrl + N/W # 新建/删除一个编辑器（文件窗口）

Ctrl + Shift + N/W # 新建/删除一个VSCode程序

Ctrl + F2 # 全选（Sublime是Alt + F3 ， 勿混淆）

Alt + left # 选取多行

Ctrl + 1 / 2 / 3 # 切换编辑器的三个视图（默认最多三个）
```

其他常用的都和Sublime差不多~~~
